# 🌙 Tumblr Themes Collection

Welcome to my repository! This is a collection of custom-coded Tumblr themes designed for various aesthetics and layouts. Feel free to browse, experiment, and make your blog look exactly how you want it.

---

## 🚀 How to Install

Follow these steps to get a theme up and running on your blog:

1. **Pick a Theme:** Open the folder of the theme you like and click on the `.html` or `.txt` file.
2. **Get the Code:** Click the **"Raw"** button in the top right of the file view.
3. **Copy Everything:** Press `Ctrl+A` (or `Cmd+A`) and then `Ctrl+C` (or `Cmd+C`) to copy the entire code.
4. **Tumblr Settings:** Go to your Tumblr Dashboard > **Edit Appearance** > **Edit Theme**.
5. **Paste & Save:** Click **Edit HTML**, delete everything currently there, and paste my code in. Hit **Update Preview** and then **Save**.

## 🛠️ Troubleshooting

* **Broken Layout?** If things look "off," check the **Theme Options** in the sidebar. Sometimes a feature (like a slider or a specific grid) needs to be toggled on.
* **Images not showing?** Make sure your image links are `https` and not `http`.
* **Report a Bug:** If you find a genuine bug in the code, please [me](https://whereinmnl.tumblr.com/).
